# Status Report — Anaplan Engagement

Weekly snapshot for the engagement leadership and client sponsor.

## Sections

- RAG status (red/amber/green) with one-line rationale
- Progress this week (bullet)
- Plan for next week (bullet)
- Risks, Assumptions, Issues, Dependencies (RAID summary; link to RAID log)
- Decisions needed from client this week
- Burn metrics (hours used vs. budget, milestone variance)
- Workstream-level status (Architecture / Build / Data / Test / Change)

## How we use it

- Distributed end-of-week, every week
- Owner: Engagement Manager / PMO
- Reviewed live at the weekly steering committee
